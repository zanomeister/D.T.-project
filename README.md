## D.T.-projekt

In folder Kaggle Mini Case you will find jupyternotebooks from a Kaggle Mini Case by Alexis Cook: https://www.kaggle.com/learn/intermediate-machine-learning. Each notebook contains:

1. [Introduction](https://github.com/zanomeister/D.T.-project/blob/master/Kaggle%20mini-course/1.%20Exercise_Introduction.ipynb)
2. [Missing Values](https://github.com/zanomeister/D.T.-project/blob/master/Kaggle%20mini-course/2.%20Missing%20values.ipynb)
3. [Categorical Variables](https://github.com/zanomeister/D.T.-project/blob/master/Kaggle%20mini-course/3.%20Categorical%20Variables.ipynb)
4. [Pipeline](https://github.com/zanomeister/D.T.-project/blob/master/Kaggle%20mini-course/4.%20Pipelines.ipynb)
5. [Cross Validation](https://github.com/zanomeister/D.T.-project/blob/master/Kaggle%20mini-course/5.%20Cross-Validation.ipynb)
6. [XGBoost](https://github.com/zanomeister/D.T.-project/blob/master/Kaggle%20mini-course/6.%20XGBoost.ipynb)
7. [Data Leakage](https://github.com/zanomeister/D.T.-project/blob/master/Kaggle%20mini-course/7.%20Data%20Leakage.ipynb)
